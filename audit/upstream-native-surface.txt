Implementation notes from the native C++ sources
- The extension registers all six exception classes from `kiwisolver.exceptions` and exposes them through the native solver methods. (source-derived)
- `Constraint`, `Term`, and `Expression` implement rich comparisons for `==`, `<=`, and `>=` only; `!=`, `<`, and `>` are unsupported and should raise `TypeError`. (source-derived from `py/src/{constraint,term,expression}.cpp`; exact message text needs release verification)
- Strength parsing accepts the four named strings and numeric values; invalid strength strings raise `ValueError`. (source-derived from `py/src/util.h`; exact message text needs release verification)
- `Solver.__new__` rejects constructor arguments. (source-derived from `py/src/solver.cpp`)
- `Variable` supports an optional context object stored separately from the native solver variable. (source-derived from `py/src/variable.cpp`; exact GC/descriptor behavior needs release verification)
